## Repository Architecture & Document Matrix

The platform is constructed utilizing a highly optimized, responsive multi-page web design styled via Tailwind CSS and fluidly animated via Animate.css. Below is the active directory mapping:

| File Name | Operational Directive Title | Code Index | Description |
| :--- | :--- | :--- | :--- |
| `index.html` | **Main Command Portal** | `PORTAL-01` | Central dashboard navigation hub for all units. |
| `policy.html` | **Manual of Policy & Procedures** | `MPP-2026` | Chain of command hierarchies, radio rules, traffic stop contact scripts. |
| `handbook.html` | **Employee Handbook** | `EHB-2026` | Core mission statements, shift hour quotas, and grooming standards. |
| `discipline.html` | **Disciplinary Actions & Procedures** | `DPCS-2026` | Level 1–3 violation categories, progressive discipline scales, and IAB appeals. |
| `emergency.html` | **Emergency Response Protocol** | `ERP-2026` | Active Shooter protocols, scene containment lines, and NIMS alignment. |
| `uof.html` | **Use of Force Directive** | `UOF-2026` | Objective reasonableness scales, resistance classifications, and restrictions. |
| `dis.html` | **Deputy-Involved Shooting** | `DIS-2026` | Immediate scene security phases, weapon surrenders, and PSS directives. |
| `bwc.html` | **Body Worn Camera Policy** | `BWC-2026` | Software rules (Medal, OBS, ShadowPlay) and end-of-shift logging mandates. |
